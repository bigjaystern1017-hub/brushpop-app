# BrushPop

  A mobile-first kids' toothbrush timer PWA. A 2-minute countdown pops away bubble overlays to reveal a hidden parent-uploaded photo.

  ## Features
  - Per-kid profiles with custom surprise images
  - 160 animated bubble overlay that pops away during brushing
  - Fanfare celebration screen with tap-to-reveal mechanic
  - Brushing streak tracking
  - All data stored in localStorage — no backend needed
  - PWA-ready

  ## Stack
  - React 18 + Vite
  - Tailwind CSS v4
  - framer-motion
  - wouter (routing)

  ## Getting Started

  ```bash
  npm install
  npm run dev
  ```

  ## Deploy to Vercel
  Connect this repo to Vercel. Build command: `npm run build`. Output dir: `dist`.
